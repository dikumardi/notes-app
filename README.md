# Notes API

A simple backend API for managing notes using Node.js, Express, and MongoDB.

## Features

- Create notes
- Read all notes
- Update a note by ID
- Delete a note by ID

## API Endpoints

- `POST /create` – Create a new note  
- `GET /read` – Get all notes  
- `PATCH /update/:id` – Update a note by ID  
- `DELETE /delete/:id` – Delete a note by ID
